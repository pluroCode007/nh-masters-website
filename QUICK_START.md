## 🌐 NETLIFY DEPLOYMENT (3 MINUTES)

1. Go to **https://netlify.com**
2. Sign up (free)
3. Click **"Add new site"** → **"Import from Git"**
4. Select **GitHub** and **your repository**
5. Build command: `npm run build`
6. Publish: `.next`
7. Click **Deploy**
8. **DONE!** Site goes live in 2-3 minutes
